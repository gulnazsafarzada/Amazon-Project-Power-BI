Amazon Data Analysis Project(Power BI)
Bu layihə Amazon platforması üçün satış, stok və müştəri loyallığı (Churn) analizini əhatə edən genişmiqyaslı Power BI hesabatıdır. Layihə 4 əsas bölmədən (səhifədən) ibarətdir.

📊 Hesabatın Səhifələri və Analizləri
1. Xəritə Analizi (Geographical Insights)
Məqsəd: Satışların coğrafi paylanmasını izləmək.

Vizuallar: Ştatlar üzrə satışların interaktiv xəritəsi.

Filtrlər: İl, Ay, Ştat və Kateqoriya üzrə dinamik filtrasiya.

2. Satış Analizi (Sales Performance)
Time Intelligence: Ötən ilin eyni dövrü (SPLY), ötən aya görə satışın artım faizi (MoM%) və ildən-ilə böyümə (YoY%).

Top Analiz: Ən çox satış edən Top 10 şəhərin müəyyən edilməsi.

Məhsul Çeşidliyi: Satışda olan məhsul növlərinin sayının izlənilməsi.

3. Stok Analizi (Inventory Management)
Məqsəd: Mövcud stok vəziyyətini və məhsul paylanmasını idarə etmək.

KPI-lar: Total Stok sayı və ən son stok vəziyyəti.

Analitika: Kateqoriya, Ölçü və Rəng üzrə stok paylanması.

4. Churn Analizi (Customer Retention)
Bu bölmədə mürəkkəb məntiqi şərtlərlə müştəri seqmentasiyası aparılmışdır:

Çıxmış Müştəri: 01.09.2022 tarixinədək 300 gündən artıq alış-veriş etməyən müştərilər.

Yeni Müştəri: Son 100 gün ərzində ilk alış-verişini edənlər.

Daimi Müştəri: Aktiv alış-verişə davam edənlər.

Vizuallar: Ştat və kateqoriya üzrə yeni/itirilmiş müştəri sayının analizi.

🛠 Texniki Bacarıqlar (Technical Skills)
DAX: Mürəkkəb CALCULATE, SAMEPERIODLASTYEAR, DATEDIFF və şərtli (IF/SWITCH) ölçülərin yazılması.

Power Query: Məlumatların təmizlənməsi, sütunların optimallaşdırılması.

Data Modeling: Cədvəllər arası əlaqələrin (Star Schema) qurulması.
